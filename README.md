# Network Monitor

A lightweight tool for monitoring network activity and connection health.

## Features

- Real-time traffic monitoring
- Connection diagnostics
- Performance metrics
- Alert notifications

## Installation

```bash
git clone https://github.com/l1ght-man/network-monitor.git
```

## Usage

```
python network_monitor.py
```
## Files

| File               |   Purpose     |
|--------------------|---------------|
| network_monitor.py | Main script   |
| monitor.log        | Ping history  |
| alert_file.txt     | Status changes|
